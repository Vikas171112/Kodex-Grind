# Spotify-Inspired Music Player (Vanilla JS)

A fully functional Spotify-inspired music player built using HTML, CSS, and Vanilla JavaScript.

---

## Features Implemented

### Core Music Player

- Dynamic playlist rendering using JavaScript
- Play / Pause functionality
- Next / Previous controls
- Auto-play next track when current ends
- Interactive progress bar with seek support
- Highlight currently playing song
- Fixed bottom mini player with thumbnail, title, and artist
